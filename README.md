# C++ Complete Roadmap

## Introduction
This repository is designed for **beginner and intermediate Software Engineering students** who want to build strong fundamentals in C++ and progress towards real-world projects.  
It covers:

### 01_Basics
- Hello World  
- Variables & Data Types  
- Input/Output  
- Scope & Lifetime  
- Operators  

### 02_Control_Flow
- If-Else  
- Switch-Case  
- For, While, Do-While Loops  
- Nested Loops  
- Pattern Printing (Diamond, Triangle)  

### 03_Functions
- Basic Functions  
- Function Overloading  
- Default Arguments  
- Inline Functions  
- Recursion  

### 04_Arrays_and_Strings
- One-dimensional Arrays  
- Two-dimensional Arrays  
- String Basics  
- String Functions  

### 05_Pointers_and_Memory
- Basic Pointers  
- Pointer Arithmetic  
- Dynamic Memory Allocation  
- New/Delete Operators  
- Pointer to Objects  

### 06_Structures_and_Unions
- Structures  
- Unions  
- Enums  

### 07_Object_Oriented_Programming
- Classes & Objects  
- Constructors  
- Destructors  
- Static Members  
- Friend Functions  

### 08_Inheritance
- Single Inheritance  
- Multilevel Inheritance  
- Multiple Inheritance  
- Hierarchical Inheritance  
- Hybrid Inheritance  

### 09_Polymorphism
- Function Overriding  
- Virtual Functions  
- Pure Virtual Functions  
- Abstract Classes  
 

### 10_File_Handling
- File Write  
- File Read  
- File Append  


### 11_Mini_Project_Bank_Management
- Create Account  
- Deposit  
- Withdraw  
- Display Account  
- Delete Account  
- Persistent storage using File Handling  
- Pure OOP (No STL)
